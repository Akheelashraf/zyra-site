# Cursor extensions for Zyra Builds (Next.js, TypeScript, Tailwind)

Install these from the Extensions panel in Cursor for a better experience.

---

## How to install

1. In Cursor, press **Cmd+Shift+X** (or click the Extensions icon in the left sidebar).
2. Search for each extension name below and click **Install**.

---

## Recommended (install in this order)

| # | Extension name | Why |
|---|----------------|-----|
| 1 | **ESLint** (Microsoft) | Runs the project’s lint rules and shows issues in the editor. |
| 2 | **Tailwind CSS IntelliSense** (Tailwind Labs) | Autocomplete and preview for Tailwind classes (e.g. `text-zyra-primary`). |
| 3 | **Prettier - Code formatter** (Prettier) | Formats code on save so style stays consistent. |
| 4 | **Path Intellisense** (Christian Kohler) | Autocomplete for file paths in `import` statements (e.g. `@/components/...`). |

---

## Optional

- **Error Lens** – Shows errors and warnings inline on the line where they occur.
- **ES7+ React/Redux/React-Native snippets** – Short snippets for React/Next components.

---

## After installing Prettier

To format on save:

1. Open Cursor **Settings** (Cmd+,).
2. Search for **Format On Save**.
3. Turn **Editor: Format On Save** ON.
4. Search for **Default Formatter** and set it to **Prettier** if you have multiple formatters.

Your project is already set up for Next.js, TypeScript, and Tailwind; these extensions only improve editing and formatting.
